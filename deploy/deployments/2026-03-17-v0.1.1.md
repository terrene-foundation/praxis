## What was deployed

- Full Praxis backend: Session management, deliberation capture, 5-dim constraint enforcement, EATP trust chains, CO Layer 5 learning pipeline
- 29 Nexus API handlers + auth endpoint
- 5 MCP tools
- 6 CO domain configurations (COC, COE, COG, COR, COComp, COF)
- SQLite persistence (ephemeral on /tmp)

## Issues encountered

1. **v0.1.0 failed**: DataFlow API incompatibility — PyPI version uses `db._models` (dict) while local dev version uses `db.models` (list of ModelDefinition). Fixed by adding compatibility layer in `persistence/__init__.py` and `persistence/db_ops.py` that supports both APIs.

2. **v0.1.1 succeeded**: After adding `hasattr()` checks for both `db.models` and `db._models`, the container starts correctly.
